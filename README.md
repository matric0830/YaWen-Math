# YaWen-Math

Ya-Wen 的高中數學教室互動教材網站。

## 資料夾結構

```text
YaWen-Math
├── index.html
├── style.css
├── script.js
├── images
│   └── chalk-banner.png
├── lessons
│   ├── junior
```

## 首頁功能

- 左側分類選單
- 章節切換
- 教材卡片
- 教師登入後新增、隱藏、刪除與修改教材狀態
